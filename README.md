# BL4S G4Beamline Simulation

This repository contains simulation codes developed for our Beamline for Schools (BL4S) experiment proposal.

The simulations use G4beamline to study particle interactions with different target materials, particularly carbon and tungsten.

The goal is to analyze how particle showers and secondary particle production vary between light and heavy nuclei.

---

## Project Goal

Our experiment investigates how hadronic interactions differ when a particle beam interacts with targets of different atomic mass.

Targets used:
- Carbon (light nucleus)
- Tungsten (heavy nucleus)

By comparing these interactions, we aim to better understand particle shower development and secondary particle production.

---

## Simulation Setup

Beamline components:
- Beam detector (Det1)
- Carbon target (50 mm)
- Detector (Det2)
- Tungsten target (50 mm)
- Detector (Det3)

The detectors:
- Measures Incoming Beam
- Observes Secondary Particles
- Measures Neutrons
